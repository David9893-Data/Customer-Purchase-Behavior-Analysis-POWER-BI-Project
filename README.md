# Customer Purchase Behavior Analysis

## Executive Summary

This analysis of electronic sales data provides actionable insights into customer purchasing trends, payment preferences, and shipping behaviors. The findings are based on visualizations created in Power BI to enhance business decision-making.

### Overview of Findings

- **Key Metrics:**
  - Total Customer Purchases: **20K** (MTD: $376.58M)
  - Total Purchase Amount: **$4.7B** (MTD: $377M)
  - Average Purchase per Customer: **$234.82K** (MTD: $252.40K)
  - Loyalty Member Rate: **2172%** (MTD: 21.18%)

- **Purchase Trends:**
  - Smartphones and laptops dominate product categories.
  - Seasonal trends show peaks in mid-year sales.

- **Payment and Shipping Preferences:**
  - Majority of customers use Credit/Debit Cards and PayPal.
  - Standard shipping remains the preferred option.

---

## Project Background

Understanding customer purchase behavior is critical for optimizing sales strategies and improving customer experience. This project analyzes key trends to support data-driven decision-making.

### Key Objectives:

1. Analyze sales trends across months and years.
2. Identify top-performing products and categories.
3. Understand customer payment and shipping preferences.
4. Provide actionable insights for marketing and logistics.

---

## Data Structure & Initial Checks

The dataset used in this project contains the following fields:

| **Field**         | **Description**                              |
|-------------------|----------------------------------------------|
| Transaction_ID    | Unique identifier for each transaction      |
| Date              | Transaction date                            |
| Product_ID        | Identifier for each product                 |
| Product_Category  | Category of the product                     |
| Quantity_Sold     | Number of units sold                        |
| Unit_Price        | Price per unit                              |
| Total_Revenue     | Total revenue from the transaction          |
| Customer_ID       | Unique customer identifier                  |
| Region            | Geographic location of the transaction      |
